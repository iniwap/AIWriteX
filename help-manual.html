I. 快速开始

三步启动

  1. 配置大模型：进入"系统设置 → 大模型API"，选择平台并填入 API Key。推荐 Deepseek（性价比高）或 OpenRouter（免费额度）。
  2. （可选）配置平台账号：如需发布到微信/头条等，在"系统设置 → 平台账号"中完成授权。建议安装 Chrome 浏览器以确保正常使用。
  3. （可选）配置配图：如需 AI 自动配图，在"系统设置 → 图片生成 API"中配置配图 API。
💡 提示：除大模型 API 外，其余配置项均有默认值，初次使用可跳过。配图 API 推荐阿里云百炼或 字节火山引擎，成本低效果好。所有设置修改后需点击"保存设置"才能永久生效。

🌐 自动热搜

输入你的主题，系统基于该主题搜索资讯并创作。适合有明确方向的创作。

II. 创意工坊

所有文章都在这里生成。工具栏提供以下功能按钮，点击展开对应设置面板。

📝 话题与写作指导

输入创作主题，留空则自动使用热搜话题。展开"写作指导"可输入具体要求、立意或风格要求，越详细文章质量越高。

🎯 赛道

选择专家赛道（如健康养生、财经投资等），AI 将按该领域的专业标准创作。可选择赛道来源、具体赛道和平台倾向，还可自定义目标读者。详见"专家赛道"章节。

📐 对标 互斥：维度化创意

提供参考文章，AI 模仿其风格和结构。支持文库、URL链接、文本粘贴三种参考来源。详见"对标模式"章节。

✨ 去AI味

自动优化AI写作痕迹。两种模式：自动模式（轻度/标准/深度优化）和协作模式（快速复核/引导式编辑/全文工作台）。详见"去AI味"章节。

🖼️ 配图 依赖：配图API

自动为文章生成封面和插图。可设置封面控制（跟随系统/固定封面/跳过）、画幅、内文配图数量和视觉风格。详见"AI配图"章节。

🌐 平台

单次覆盖系统级平台设置，优先级最高。可选择发布平台、文章格式、目标字数、发布类型（图片消息/普通图文）。详见"单次创作覆盖"章节。

⚠️ 重要互斥关系：
• 对标模式与维度化创意（系统设置→创意设置）不能同时开启——二者都会深度重写内容，叠加会导致不可预期结果。
• 专家赛道可与对标模式搭配使用，但当赛道内容与对标参考不一致时，赛道仅作为框架约束。

IV. 对标模式

提供参考文章，让 AI 模仿其风格、结构和排版。

配置步骤

STEP 1 · 参考来源

选择参考内容的来源方式：文库（从微信文库选择已导入文章）、URL（粘贴网页链接，多个用竖线 | 分隔）、或文本（直接粘贴 Markdown 或纯文本）。

STEP 2 · 对标方式

选择改写程度，从轻到重：原文直用（成套模板）→ 轻改（轻量润色）→ 影写（保留推进重写）→ 降重（降低相似度）→ 重构（重组结构）→ 仿写（只学结构）→ 融合（多文去重合成）。

STEP 3 · 精细参数

可选调整：事实边界（严格锁定/锁核心/允许补背景）、案例处理（重写/换新/只留场景）、信息补充（不补充/补背景/补论据）。

⚠️ 注意：
• 对标模式与维度化创意（系统设置→创意设置）不能同时开启。
• 对标模式下，专家赛道的语气风格约束力会减弱，AI 会优先模仿参考文章风格。
• URL 参考链接必须是可公开访问的网页。微信文章建议先导入文库再使用。
• 对标模式下，"事实边界"设为"严格锁定"时，AI 不会添加参考文章以外的新信息。

V. 模板复刻教程

从一篇微信公众号文章，到完全复刻其排版模板并应用到自己的内容——完整流程。

STEP 01 · 导入文章

微信文库导入

  1. 进入 内容中心，切换到微信文库面板
  2. 点击 "导入链接"，粘贴微信公众号文章链接
  3. 等待系统自动抓取文章内容和样式
STEP 02 · 创建模板

清洗并保存模板

  1. 进入 模板管理 → 新建模板
  2. 在弹窗中 从微信文库选择 刚导入的文章
  3. 勾选 "清洗HTML"（重要！微信有大量冗余代码，建议清洗）
  4. 填写模板名称，点击确定。清洗约需 1-2 分钟
STEP 03 · 应用模板

使用模板生成文章

  1. 进入 创意工坊，开启对标模式
  2. 选择刚创建的模板，对标方式设为 原文直用
  3. 参考来源选择 "文本"，粘贴你的文章内容
  4. 点击 "开始生成"

系统将用你的内容，套用复刻的模板样式，生成视觉效果一致的新文章。

💡 技巧：模板复刻 + 对标模式 → 原文直用，是将已有内容快速套用精美排版的最快方式。适合需要统一品牌视觉的公众号运营。

VII. AI 配图 & 资源图库

AI 配图怎么用

  1. 配置配图 API：系统设置 → 图片生成 API，选择模型并填入 API Key。
  2. 创意工坊开启：点击"配图"按钮展开面板，设置封面（跟随系统/固定/跳过）和内文配图数量。
  3. 开始生成：系统自动分析文章内容，生成匹配的插图并插入对应位置。

封面与内文配图设置

  • 封面控制：跟随系统（根据设置自动生成）、使用固定封面（从图库选择或本地上传）、跳过封面。
  • 封面画幅：支持 2.35:1（微信封面）、16:9、9:16、4:3、3:2、1:1 等多种比例。
  • 内文配图：可设最大配图数（1-10张）、视觉风格和数据墨水比。
  • 配图模式（系统设置中）：AI 自动、仅图库匹配、跳过配图。

资源图库

  • 管理所有 AI 生成和上传的图片，支持分类管理、标签搜索、按来源筛选。
  • 图库优先：开启后优先从图库匹配已有图片，避免重复生成浪费额度。
  • Prompt 中心：查看和管理图片的 Prompt 模板资源。
  • 实验区：独立于文章配图，直接输入 Prompt 生成和调试图片效果。
⚠️ 注意：
• 配图是可选功能。不配置配图 API 时，系统使用图库匹配或跳过配图。
• 配图生成消耗 API 额度，建议在实验区调试好 Prompt 后再用于批量配图。
• "图库匹配阈值"可调节匹配精度，阈值越高匹配越严格。
• 推荐使用付费模型（如 Google Nano Banana）获得更好效果，免费模型效果有限。

IX. 文章格式与输出

理解格式差异，确保文章在各平台正确显示。

三种输出格式

HTML — 带排版样式，支持模板、CSS 样式、图文混排。
设置路径：系统设置 → 基础设置 → 文章格式
Markdown — 纯文本标记语言，排版由平台渲染。
适合不需要复杂排版的内容。
TXT — 纯文本，无任何格式标记。

各平台格式支持情况

平台HTML模板说明
微信公众号✓✓完整支持 HTML 排版和模板
百家号✓✓完整支持 HTML 排版和模板

微博✓✗支持 HTML，不支持模板样式
B站✓✗支持 HTML，不支持模板样式

模板与格式的关系

  • 启用模板时：输出自动为 HTML 格式（模板 = HTML 排版）。适合微信公众号和百家号。
  • 不启用模板时：输出格式取决于基础设置中的"文章格式"选项。
  • 自动降级：启用模板但目标平台不支持 HTML 时，系统会自动降级为纯文本，不使用模板设置。
⚠️ 关键提醒：如果目标平台不支持 HTML（如知乎、小红书等），开启模板不会报错但模板样式会被忽略。建议关闭模板功能或将文章格式设为 Markdown/TXT。

X. 图片消息与发布类型

创意工坊的"平台"面板中，可以选择两种发布类型，适用于不同平台和场景。

两种发布类型

📷 图片消息

以图片形式发布，适用于公众号贴图、小红书图文等场景。两种子模式：

  • 配图模式：AI 提炼文章要点并生成配图，配文建议不超过 300 字。
  • 长图模式：将全文渲染为一张长图截图发布。
📄 普通图文

以传统文章形式发布。两种子模式：

  • 正常文章：标准图文发布。
  • 长图模式：全文截图后以图片形式发布，保留完整排版。

注意事项

  • 图片消息 + 配图模式：配文字数建议≤300字，系统会自动截断过长内容。
  • 图片消息 + 长图模式：文章固定保存为 TXT 格式。
  • 普通图文 + 长图模式：文章固定保存为 HTML 格式以保留排版。
  • 图片消息分类：发布到头条时可选择内容分类（国际、体育、娱乐等 30+ 分类）。
💡 提示：创意工坊"平台"面板中的设置会覆盖系统设置中的平台配置，优先级最高。勾选"跟随系统"则使用系统设置的默认值。

XI. 草稿 · 定时 · 批量

内容中心（草稿箱）

所有生成的文章自动保存在内容中心。三种状态：未发布 / 已发布 / 发布失败。

  • 支持可视化编辑和 HTML 源码编辑。
  • 支持批量选择文章统一发布或删除。
  • 发布失败的文章会保留错误原因，方便排查。
  • 包含微信文库面板，可管理导入的微信文章素材。

定时任务 高级

  1. 进入 定时任务，点击"新建任务"。
  2. 设置发布时间、数量、生成参数（话题、赛道、模板等）。
  3. 也可选择"使用已有文章"，直接定时发布，无需重新生成。
  4. 任务支持 启动/暂停/取消/删除 操作。

批量生成 高级

  • 一次生成多篇文章，适合大量内容储备。
  • 设置全局参数后，可为单篇独立覆盖设置。
  • 生成完成后可在内容中心统一管理和发布。
💡 技巧：批量生成 + 定时发布搭配使用，可一次性准备好一周的内容并自动发布。

XII. 小说连载 高级

从世界观设定、角色管理到章节创作，完整的小说创作工作流。支持卷结构管理、AI 大纲规划和章节自动生成。

操作流程

  1. 创建项目：点击"新建小说"，填写书名、分类、目标字数，选择是否启用卷结构。
  2. AI 初始化：系统自动生成世界观设定、核心角色和初始大纲。
  3. 规划大纲：点击"AI 智能规划"生成全书大纲。注意：开始写作后将无法重新生成大纲。
  4. 创作章节：选择章节，使用"AI 编写"自动生成内容。支持"扩展角色"功能按需增加新角色。

注意事项

  • 大纲不可逆：一旦开始写作（创建章节），将无法重新生成大纲。如需调整，请删除所有章节后重试。
  • 卷结构：中长篇建议启用，短篇可选择不启用，章节平铺显示。
  • 角色管理：角色分主角、配角、反派三种类型，AI 据此设定行为逻辑。
  • 渐进式生成：建议初始只生成核心角色和少量章节大纲，后续按需扩展。

XIII. 去AI味 & 数据洞察

去AI味（自然表达优化）

消除文章的"AI机械感"，让内容更像真人创作。

  • 自动模式：AI 自动分析并改写，三个等级——轻度优化（保留更多原文，速度最快）、标准优化（自然度与效率兼顾，推荐）、深度优化（表达最自然，耗时稍长）。
  • 协作模式：改写后保留人工审核环节，三个等级——快速复核（展示残留建议，一键确认）、引导式编辑（展示复核点，按需微调）、全文工作台（全文编辑，对照关键复核点）。
  • 文风参考：可粘贴一篇非AI文章作为文风模仿样本，AI将模仿其语气和句式节奏。
  • 高级设置：可微调自然度提升、语言模式化、结构规整度、情感色彩等权重参数。
💡 建议：去AI味与对标模式可同时使用——先对标模仿风格，再去AI味优化表达。

XIV. 内容增效与自适应进化

系统设置 → 基础设置中提供两组进阶功能，可显著提升内容质量和系统智能度。

内容增效与防坑

  • 自媒体爆款滤镜：强制文章以痛点场景开头、互动引导结尾，适合微信/小红书等营销平台。开启后文章更具传播力，但可能影响内容的客观性。
  • 深度防坑与洞察：强制文章使用具体数据和逻辑骨架，杜绝AI常见的空泛修饰和"正确的废话"。适合需要高质量内容的场景。
  • 人工数据校验占位：当 AI 缺乏真实数据支撑时，输出占位符而非编造数据，由您人工补充填空。适合对数据准确性要求高的内容。如您只需要一键全自动生成，请关闭此项。
⚠️ 互斥提示：自媒体爆款滤镜和深度防坑可以同时开启，但二者的写作目标不同（一个追求传播力，一个追求严谨性），可能对文章风格产生相互制约的影响。

自适应进化

  • 应用稳定参数进化：允许系统根据累计评分自动微调去AI味强度等数值型配置，使系统越用越贴合您的偏好。
  • 应用实验性系统守则：允许系统分析用户评分和建议，自动生成提示词守则来优化后续创作。
  • 启用 AI 深度评分：调用大模型对文章进行真实打分（会消耗 Token）。关闭则只使用本地快速评分，不消耗额度但评分精度较低。
💡 进化机制说明：进化功能依赖于用户评分数据。建议每次创作后给文章打分（在数据洞察中心或文章预览中），评分越多系统优化越精准。开启 AI 深度评分能获得更准确的分数，但每次评分会消耗少量 Token。

XVI. 微信 API 中继

针对家用宽带或移动办公用户，IP 地址频繁变动导致微信后台白名单配置繁琐。 启用中继后，微信 API 请求会通过您自建的固定 IP 服务器转发，只需将中继服务器 IP 加入白名单即可。 中继只转发微信官方 API，不具备通用代理能力。

适用场景

  • 家用宽带无固定公网 IP，每次拨号 IP 都变。
  • 移动办公，经常切换网络环境。
  • 不想频繁登录微信后台修改 IP 白名单。

如果您使用代理合作方提供的版本，微信中继由服务端统一下发和管理，无需自行搭建。

服务端部署

在具备固定公网 IP 的云服务器上，创建 wechat_relay_server.py 并运行：

XVIII. 常见问题

Q. 生成文章需要多久？

通常 3-5 分钟，取决于是否开启配图和去AI味。

Q. 文章没有自动发布？

检查：① 基础设置中"自动发布"是否打开？② 基础设置中"发布平台"是否勾选？③ 微信：IP 是否在白名单（或用中继）？④ 其他平台：登录授权是否有效？

Q. 如何提高文章质量？

① 写详细的写作指导 ② 选择合适的专家赛道 ③ 提供高质量参考文章（对标模式）④ 组合使用去AI味 ⑤ 开启"深度防坑与洞察"。

Q. 对标模式和维度化创意能同时用吗？

不能。二者都会深度重写内容，叠加会产生不可预期的结果。建议单独使用。

Q. 启用模板后发到知乎/小红书为什么没有排版？

知乎、小红书等平台不支持 HTML，系统会自动降级为纯文本。如需在这些平台发布，建议关闭模板或使用 Markdown 格式。

Q. 创意工坊里的设置和系统设置冲突时以哪个为准？

创意工坊"平台"面板中的设置优先级最高，会覆盖系统设置中的发布平台和文章格式。勾选"跟随系统"则使用系统默认值。

Q. AI 配图必须配置吗？

不是必须。不配置配图 API 时，系统会尝试从图库匹配已有图片或跳过配图。配图是可选增强功能。

Q. 平台登录授权过期了怎么办？

在系统设置 → 平台账号中重新登录即可。

Q. 图片消息和普通图文有什么区别？

图片消息以图片形式发布（适合公众号贴图、小红书图文），普通图文以文章形式发布。详见"图片消息与发布类型"章节。

Q. 微信 API 中继有什么用？

解决家用宽带/移动办公 IP 频繁变动导致微信后台白名单配置繁琐的问题。详见"微信 API 中继"章节。

Q. 哪些功能需要高级版？

小说连载、批量生成、定时任务为高级版功能，需升级后才能使用。
