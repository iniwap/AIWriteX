VII. 去AI味功能

AI生成的内容往往带有明显的"机械感"，容易被读者察觉。去AI味功能通过智能改写，让文章更加自然、人性化。

A

功能入口

  • 创意工坊：“去AI味”，选择强度开始生成即可
  • 创意工坊：文风参考用于模仿非AI文章的文风
  • 创意工坊：对本次“去AI味”的结果进行“打分”，以优化算法模型
B

处理原理

系统会分析文章中的AI痕迹特征，如过于工整的句式、重复的过渡词、模式化的开头结尾等，然后进行针对性改写：

AI痕迹示例

"首先...其次...最后..."、"值得注意的是"、"综上所述"等模板化表达

改写效果

更自然的口语化表达、多样化的句式结构、个性化的观点陈述

C

使用建议

  • 适用场景：需要发布到对原创要求较高的平台时使用
  • 处理时间：根据文章长度，通常需要1-3分钟
  • 效果检验：处理后建议通读全文，确保语义连贯

VIII. 模板复刻教程

"从一篇微信公众号文章链接，到完全复刻其模板并应用到自己的内容" —— 完整流程指南

STEP 01 · 导入文章

微信文库导入

  1. 进入 内容中心 → 微信文库
  2. 点击 "导入链接" 按钮
  3. 粘贴微信公众号文章链接
  4. 等待系统自动抓取文章内容和样式
STEP 02 · 创建模板

清洗并保存模板

  1. 进入 模板管理 → 新建模板
  2. 在弹窗中 从微信文库选择 刚导入的文章
  3. 勾选 "清洗HTML"（重要！微信有大量垃圾代码，建议清洗）
  4. 填写模板名称，点击确定等待清洗完成
⚠️ 提示： 清洗过程需要1-2分钟，请耐心等待。清洗后的模板更精简，渲染效果更好。
STEP 03 · 应用模板

使用模板生成文章

  1. 进入 创意工坊
  2. 开启 借鉴模式，选择刚创建的模板
  3. 将借鉴比例设置为 100%（直接使用内容）
  4. 在参考来源选择 "文本粘贴"，粘贴您的文章内容
  5. 点击 "开始生成"

🎉 完成！
系统将用您提供的内容，套用复刻的微信模板样式，生成一篇视觉效果完全一致的新文章。

XII. 微信代理转发

针对家用拨号上网或移动办公用户，IP 地址频繁变动会导致微信后台白名单配置极为繁琐。 代理转发功能允许您通过一台固定 IP 的中转服务器发送发布指令，彻底解决此痛点。

1

准备工作

您需要准备一台具备固定公网 IP 的云服务器。推荐选择阿里云、腾讯云或华为云的“轻量应用服务器”，每月成本极低，且能提供极其稳定的固定出口 IP。

2

服务端代码 (Python 示例) 

# 环境准备: pip install flask requests
from flask import Flask, request
import requests

app = Flask(__name__)

@app.route('/proxy', methods=['POST'])
def proxy():
    url = request.args.get('url')
    if not url: return "Missing URL", 400
    
    # 透传请求头、数据与方法
    headers = {k: v for k, v in request.headers if k.lower() != 'host'}
    try:
        response = requests.request(
            method=request.method,
            url=url,
            headers=headers,
            data=request.get_data(),
            timeout=10
        )
        return (response.content, response.status_code, response.headers.items())
    except Exception as e:
        return str(e), 500

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5000)
3

AIWriteX 配置

  1. 将上述代码在云服务器部署运行（建议使用 screen 或 pm2 持久化）。
  2. 在微信公众号后台将该服务器固定 IP 加入 IP 允许列表（白名单）。
  3. 在 AIWriteX 系统设置 -> 微信公众号 中勾选“启用代理转发”。
  4. 填入中转地址，例如：http://1.2.3.4:5000/proxy。

XV. 小说连载

提供了从项目创建、世界观设定、角色管理、章节创作到发布的完整工作流。支持卷结构管理、AI智能大纲规划、章节自动生成等功能，让小说创作更加高效。

标准操作工作流

  • 创建项目：点击"新建小说"，填写书名、分类、目标字数，选择是否启用卷结构。 支持手动填写或AI智能构建模式。
  • AI初始化：系统会自动生成世界观设定、核心角色和初始大纲。
  • 生成全局大纲：点击"AI智能规划"生成完整的全书大纲。 注意：此操作仅在未开始写作时可用，一旦有章节内容将无法重新生成。
  • 管理卷结构：小说大纲面板，编辑可在左侧边栏创建和管理卷，每卷包含若干章节。
  • 创作章节：选择或创建章节后，在右侧编辑器中编写内容。 可使用"AI自动规划"功能自动生成章节。
  • 编写章节：选择章节，在右侧编辑器中编写内容。 可使用"AI编写"功能自动生成内容。
  • 扩展角色：随着故事发展，可使用"扩展角色"功能AI自动生成新角色。 AI会参考已有角色设定，保持风格一致。
  • 手动大纲编辑：大纲生成后，可点击编辑按钮手动修改内容。 保存后将覆盖原有大纲。

字数对创作的影响

  • 3万字：1卷结构，约15-20章，单一线索，紧凑篇幅
  • 10万字：2卷结构，约30-40章，主线+1-2条辅线
  • 30万字：3-4卷结构，约60-80章，多线叙事
  • 50万字：5-6卷结构，约100-120章，宏大世界观
  • 100万字：8-10卷结构，约200-250章，史诗级巨著

重要：目标字数会深度影响AI生成的大纲结构、世界观复杂度和角色数量。请在创建项目时谨慎选择。

注意事项

  • 大纲不可逆：一旦开始写作（创建章节），将无法重新生成大纲。 如需重大调整，请删除所有章节后重试。
  • 卷结构选择：卷结构与字数为独立选项。中长篇建议启用卷结构，便于章节管理； 短篇可选择不启用，所有章节平铺显示。
  • AI初始化耗时：首次AI初始化需要生成世界观、角色和大纲，耗时较长（1-2分钟）。 后续扩展角色或章节会相对较快。
  • 角色类型说明：角色分为"主角(MAIN)"、"配角(SUPPORTING)"和"反派(VILLAIN)"三种。 AI生成时将据此设定角色的行为逻辑和成长弧线。

高端使用技巧

  • 渐进式生成：初始只生成2个核心角色+3章大纲，后续通过"扩展角色"和"扩展章节" 功能按需扩展，保持创作灵活性。
  • 手动大纲微调：AI生成大纲后，可手动编辑调整细节。点击标题旁的编辑图标即可打开编辑器。
  • 卷章节联动：大纲生成时会自动解析卷标题并创建对应卷结构。确保大纲中使用"卷一：标题" 或"第1卷：标题"格式。
  • 情绪曲线规划：大纲中的每个章节都应有明确的情绪走向（爽点/低谷）， 这将帮助AI在续写时保持情节张力。
  • 世界观迭代：随着故事发展，可随时回到"设定集"标签页更新世界观、势力和道具设定。

XVI. 常见问题

Q. 生成文章需要多长时间？

通常需要 5 分钟左右。具体取决于话题的复杂度和HTML模板适配的耗时。

Q. 文章为何没有自动发布？

  • 公众号 AppID/Secret 是否填写正确？
  • 是否开启了“自动发布”开关？
  • 服务器 IP 是否已加入公众号后台白名单？
  • 公众号是否认证（未认证只能发布到草稿箱）？

Q. 如何提升文章质量？

  • 在创意工坊的写作指导里输入更详细的文章要求
  • 通过热搜雷达的话题分析功能，生成写作指导，提示质量；
  • 建议开启借鉴模式并提供高质量参考文；
  • 或者尝试组合使用维度化创意功能。
  • 借鉴模式和创意组合建议单独使用，二者有一定的冲突。

Q. 支持哪些平台发布？

目前主要支持微信公众号，其他平台正在开发中。
