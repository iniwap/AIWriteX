🚀 AIWriteX - 微信公众号自动生成工具

AIWriteX 是一款结合 AIPy 与 CrewAI 的强大工具，可自动获取抖音、微博等热点话题，并借助 AI 实时生成高质量图文内容，支持酷炫排版并一键发布到微信公众号。

📌 功能亮点

  • 📈 实时热点获取
  • 🤖 AI 高质量写作
  • 📤 一键图文发布
  • 🧩 支持多种模板 & 图文管理

🛠️ 快速开始

git clone https://github.com/iniwap/AIWriteX.git
pip install -r requirements.txt
python main.py -d
