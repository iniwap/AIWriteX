常见 问题

生成文章需要多久？

通常需要2-5分钟。系统需要实时搜索网络资讯、分析内容并进行AI深度创作，好内容值得等待。

支持哪些大模型？

目前完美支持 DeepSeek、OpenRouter 等主流接口，您可以在设置中自由切换和配置 API Key。

如何购买授权码？

点击“获取授权码”跳转至淘宝店铺购买。购买后打开软件，输入“订单+手机尾号后4位”即可激活。

更换电脑怎么办？

授权码与机器绑定。如需更换设备，请联系售后客服进行解绑重置（高级版用户享有此服务）。
